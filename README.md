# Getting Started

### How To Use - Guide
* Clone or download the zip.
* Open it using any IDE. (I have used my favourite IDE - Intellij)
* Run the main Spring Boot Application file - UrlShortenerApplication
* By default it be will hosted on Http port 8080
* Access the app using http://localhost:8080/
